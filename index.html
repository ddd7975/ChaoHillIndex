ChaoHill

  • Author: K. S. Tseng ; Y. H. Lee ; Anne Chao
  • Contributor: K. S. Tseng
  • URL: http://chao.stat.nthu.edu.tw/blog/

The purpose of this R package ChaoHill is mainly to provide Hill's number curve for both individual-based (abundance) data and sample-based (incidence) data.

Run the development version

  • Required: R
  • Suggested: Rstudio

Start R(studio) and copy-and-paste the commands below:

#install.packages('devtools')
library(devtools)
install_github('ChaoHill','ddd7975')
library(ChaoHill)
